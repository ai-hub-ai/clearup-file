# ClearUp File 🧹

**ClearUp File** 是一款现代化的跨平台大文件清理工具，专为帮助用户快速释放磁盘空间而设计。它结合了高性能的文件扫描引擎与智能化的数据分析界面，让磁盘清理变得安全、直观且高效。

## ✨ 核心特性

### 1. 🚀 高性能全盘扫描
- **极速引擎**：基于 Node.js 流式处理，快速遍历深层目录。
- **灵活配置**：支持自定义扫描目录和最小文件大小阈值（如 >100MB, >1GB）。

### 2. 📊 智能可视化分析
- **文件分布图谱**：自动将文件分类为 **视频、音频、图片、文档、归档、安装包** 等，通过彩色进度条直观展示各类文件占用比例。
- **实时统计**：侧边栏实时更新各类文件的数量与总大小。

### 3. 🛡️ 智能清理建议 (AI-Assisted)
独创的智能分级系统，保护您的重要数据：
- 🟢 **建议清理**：高置信度的垃圾文件（如 `Downloads` 中的安装包、`Caches` 缓存、`node_modules` 等）。
- 🟡 **谨慎清理**：用户个人数据（如 `Documents`、`Desktop`、应用数据），需用户确认。
- 🔴 **系统文件**：严格保护系统关键路径（如 `/System`、`/Library`），防止误删导致系统崩溃。

### 4. 🎨 现代化交互体验
- **主题切换**：内置 **深色 (Dark)** 和 **浅色 (Light)** 两种精美主题，一键切换。
- **交互式筛选**：点击侧边栏的分类或建议标签，即可快速过滤文件列表。
- **多维度排序**：支持按大小、修改时间、路径进行排序。
- **CSV 导出**：支持将扫描结果导出为 CSV 表格，便于进一步分析。

### 5. ⚡️ 便捷文件管理
- **批量操作**：支持多选、全选文件。
- **安全删除**：默认“移至回收站”，提供后悔药；也支持彻底删除。
- **文件移动**：一键将大文件归档移动到外部硬盘或其他目录。
- **快捷键支持**：`Space` (预览 - 规划中), `Enter` (打开), `Del` (删除), `Cmd/Ctrl+A` (全选)。

## 🛠 技术栈

本项目采用最新的前端与桌面端技术构建：

- **核心框架**: [Electron](https://www.electronjs.org/) (跨平台桌面运行时)
- **UI 框架**: [React 18](https://reactjs.org/) (组件化用户界面)
- **构建工具**: [Vite](https://vitejs.dev/) (极速构建与热更新)
- **样式**: CSS3 Variables (主题化) + Flexbox (响应式布局)

## 💻 开发与运行

### 环境要求
- Node.js (v16+)
- npm 或 yarn

### 快速开始

1. **克隆项目**
   ```bash
   git clone https://github.com/your-repo/clearup-file.git
   cd clearup-file
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **开发模式启动**
   同时启动 React 开发服务器和 Electron 主进程：
   ```bash
   npm start
   ```

4. **构建生产版本**
   ```bash
   npm run build
   ```
   构建产物将位于 `dist` 或 `release` 目录中。
